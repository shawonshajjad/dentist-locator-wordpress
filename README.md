# Dentist Locator with Interactive SVG Map

A custom WordPress plugin that allows users to find dental practices via an interactive map of Australia or a live AJAX search bar.

## 🚀 Key Features
* **Custom Post Type:** Dedicated "Dentists List" management.
* **Interactive SVG Map:** Clickable Australian states for quick filtering.
* **Live AJAX Search:** Real-time results by Name, Postcode, or Category.
* **Admin Settings:** Customizable "No Results" message and contact details.

## 🛠️ Technical Details
* **Backend:** PHP, WordPress Settings API, Metadata API.
* **Frontend:** Vanilla JavaScript (AJAX), CSS3 Flexbox/Grid.
* **Structure:** Organized assets folder for maintainability.
